# Zenith-Pomodoro-Timer
Zenith Pomodoro Timer is a productivity tool designed to boost focus and efficiency using the proven Pomodoro Technique. Organize work into focused sessions followed by short breaks, helping you stay motivated, reduce distractions, and achieve more with better time management.

# Zenith Pomodoro Timer

Zenith Pomodoro Timer is a refined browser-based productivity app built with Tailwind CSS and vanilla JavaScript. It delivers an immersive focus workflow with Pomodoro timing, surround-sound ambience, task tracking, analytics, and customizable session settings.

## Features

- Pomodoro work timer with short and long break cycles
- Multiple timer modes: Pomodoro, Short Break, Long Break
- Ambient audio engine with binaural focus hum, rain, and celestial drift
- Tick pulse audio and session completion chime
- Task list with completion toggles and persistent local storage
- Session history logging and analytics dashboard
- Settings panel for custom Pomodoro durations and auto start behavior
- Dark mode support and responsive modern UI
- Animated 3D particle background and interactive timer card tilt

## Built With

- HTML
- Tailwind CSS (CDN)
- JavaScript
- Lucide icons

## Project Structure

- `index.html` — main application UI
- `style.css` — custom styling and visual polish
- `app.js` — timer engine, audio, tasks, analytics, and local storage

## Usage

1. Open `index.html` in a modern browser.
2. Select a session mode: Pomodoro, Short Break, or Long Break.
3. Start or pause the timer using the central control.
4. Add tasks and mark them complete during your session.
5. Open settings to adjust durations, auto-break behavior, and ticking sound.
6. View session history and progress analytics in the dashboard.

## Notes

- The app runs entirely in the browser; no installation required.
- All user settings, tasks, and history are saved locally with `localStorage`.
- For the best experience, use a browser that supports the Web Audio API.
